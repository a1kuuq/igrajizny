# Игра Жизнь (WinForms)

Простое WinForms-приложение для лабораторной работы по клеточной модели «Игра Жизни» Дж. Конвея. Реализованы последовательный и параллельный расчёт следующего поколения с возможностью отмены вычислений.

## Запуск
1. Откройте `GameOfLife/GameOfLife.sln` в Visual Studio 2022+ на Windows.
2. Соберите и запустите проект `GameOfLife` (целевой фреймворк: `net6.0-windows`).
3. Используйте кнопки формы:
   - **Последовательно** — расчёт поколения в одном потоке.
   - **Параллельно** — расчёт с использованием `Parallel.For`.
   - **Отмена** — остановка текущей симуляции через `CancellationToken`.
4. Поле автоматически перерисовывается после каждого шага, текущий номер поколения отображается под сеткой. Регулируйте паузу между шагами через поле задержки (мс).
